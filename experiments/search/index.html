Search Algorithms in Animations

On this page I demonstrate Depth-First Search, Breadth-First Search, and A* Search.

Pathfinding algorithms are best examplified through (unsurprisinbly) pathfinding in games. In these demos a will walk through avoiding in order to reach the .

You will need javascript enabled to see demos. I drew the sprites and coded algorithms. These demos use PIXI.js for actual drawing.

Depth-First Search

It is a very simple algorithm that uses stack to keep track of unexplored nodes.

Breadth-First Search

In fact, breadth-first search is just like DFS, but using queue instead of stack.

A* Search

Most complex of these algorithms, A* uses heuristics + knowledge to select order for traversal. In this case, heuristics is Manhattan distance. WARNING: algorithm there is incorrectly implemented and thus sometimes it couldn't find the way when such a way exists.
